# Pairwise differences

Folder for code testing whether pairwise environmental distance between environments-of-origin or pairwise genetic distance (Fst) between parent populations predicts differences in survival and shell length across hatchery, nursery, and field stages.

| File | Function | Datasets used | Datasets produced |
|------|----------|---------------|-------------------|
| pairwise_differences_analyses.Rmd | Builds pairwise distance matrices for all response variables (hatchery survival at day 21, hatchery length at days 21 and 78, field length and survival at each monitoring event Ã— site combination). Runs partial Mantel tests to assess whether pairwise genetic distance (Fst) predicts performance differences after controlling for environmental distance, and vice versa. Applies BH correction for multiple testing across all tests. Includes linearity assumption checks and produces heatmaps of predictor distances (Fst, environmental distance) and response distances (survival, length) at the final field timepoint. | **Performance data:**<br>`nursery_length.csv` (from Q1_nursery_performance.Rmd), `CViMVP_larvae_survival.csv`, `field_length_all_times.csv` (from length_calc_field.Rmd), `field_surv_all_times.csv` (from survival_calc_field.Rmd)<br><br>**Environmental data:**<br>`envr_quantiles_summary.csv`, `envr_quantiles_dist.rds` (scaled Euclidean distance matrix)<br><br>**Genetic data:**<br>`20250606_fst_results.rds` (pairwise Fst matrix, from Fstats_experimental.Rmd) | **Figures (PDF):**<br>`Fig4A_fst_heatmap.pdf`, `Fig4B_envr_dist_heatmap.pdf`, `Fig4C_survival_lew_t3_heatmap.pdf`, `Fig4D_survival_york_t3_heatmap.pdf`, `FigS10A_length_lew_t3_heatmap.pdf`, `FigS10B_length_york_t3_heatmap.pdf` |
