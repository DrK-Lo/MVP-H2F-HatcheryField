# Parent effects

Folder for code related to modeling the effects of parental genetics and environment-of-origin on oyster performance (survival, length, condition index) across hatchery, nursery, and field stages.

| File | Function | Datasets used | Datasets produced |
|------|----------|---------------|-------------------|
| Q2_envr_quantile_models_field.Rmd | Models drivers of field survival and length using parental genetic diversity (Ho, allelic richness) and environment-of-origin quantiles (salinity Q10/Q90, temperature Q10) as predictors, separately by site and time point. Drops temp Q90 upfront due to collinearity (r > 0.7 with Ho and Ar). Assesses VIF for remaining variables. Uses backward stepwise selection via `drop1()` (survival lm) or `step()` (length lmer). Produces scaled coefficient slope plots across time points. | `field_surv_all_times.csv`, `field_length_all_times.csv`, `parent_genetic_effects.csv`, `envr_quantiles_allsites.csv` | **Figures (PDF):** `Fig3A_lew_surv_quantiles.pdf`, `Fig3C_york_surv_quantiles.pdf`, `FigS5A_lew_length_quantiles.pdf`, `FigS5C_york_length_quantiles.pdf`, `FigS2_corrplot_envr_quantiles_H0_Ar.pdf` · **Results (TXT):** `survival_model1_quantiles_results_<date>.txt`, `length_model1_quantiles_results_<date>.txt` |
| Q2_envrdist_models_field.Rmd | Models drivers of field survival and length using parental genetic diversity (Ho, allelic richness) and scaled Euclidean environmental distance (EoO to each common garden site) as predictors, separately by site and time point. Assesses multicollinearity and VIF for each site separately. Computes residuals of survival/length ~ Ho + Ar for downstream residual plots. | `field_surv_all_times.csv`, `field_length_all_times.csv`, `parent_genetic_effects.csv`, `envr_quantiles_dist_all.csv` | **Figures (PDF):** `Fig3B_lew_surv_envrdist.pdf`, `Fig3D_york_surv_envrdist.pdf`, `FigS5B_lew_len_envrdist.pdf`, `FigS5D_york_len_envrdist.pdf`, `FigS8A_corrplot_lew_envrdist_H0_Ar.pdf`, `FigS8B_corrplot_york_envrdist_H0_Ar.pdf` · **Results (TXT):** `survival_model2_envrdist_results_<date>.txt`, `length_model2_envrdist_results_<date>.txt` · **Data (CSV):** `residuals_distance_vs_surv.csv`, `residuals_distance_vs_length.csv` |
| Q2_condition_index_models_field.Rmd | Models drivers of field condition index using two approaches: (1) Ho/Ar + environmental quantiles, and (2) Ho/Ar + environmental distance. Uses LMMs with bag as random effect, backward stepwise selection, and scaled slope plots. Also generates the combined supplementary table of all Q2 model results (Supp File 4) by programmatically extracting estimates from all final models across the three Q2 scripts. | `MVP_phenotyping_final.csv`, `parent_genetic_effects.csv`, `envr_quantiles_allsites.csv`, `envr_quantiles_dist_all.csv` | **Figures (PDF):** `FigS7A_lew_ci_quantiles.pdf`, `FigS7B_lew_ci_envrdist.pdf`, `FigS7C_york_ci_quantiles.pdf`, `FigS7D_york_ci_envrdist.pdf` · **Results (TXT):** `condition_model1_quantiles_results_<date>.txt`, `condition_model2_envrdist_results_<date>.txt` · **Supplementary file (CSV):** `4-Q2_all_model_results.csv` (Supp File 4) |
| Q2_models_hatchery_nursery.Rmd | Models effects of parental genetic diversity and environmental quantiles on larval length (day 21 and day 77) and larval survival (day 21) in the hatchery/nursery using multiple regression. Uses backward model selection via `drop1()`. Notes that day 21 survival model has too few data points for robust inference. Produces scaled coefficient slope plot comparing hatchery and nursery stages. | `nursery_length.csv`, `envr_quantiles_allsites.csv`, `CViMVP_larvae_survival.csv`, `parent_genetic_effects.csv` | **Figures (PDF):** `FigS6_hatchery_nursery_surv_quantiles.pdf` · **Results (TXT):** `hatchery_nursery_model_results_<date>.txt` |
| Q2_envrdist_residuals_plots.Rmd | Plots survival and length residuals (after removing Ho + Ar effects) against environmental distance to each common garden site, colored by source population. Generates exploratory plots for all three timepoints but only saves final timepoint (t3) panels for the supplement. Tests whether southern populations drive the positive relationship between environmental distance and performance. | `field_surv_all_times.csv`, `envr_quantiles_dist_all.csv`, `residuals_distance_vs_surv.csv`, `residuals_distance_vs_length.csv` | **Figures (PDF):** `FigS9A_lew_surv_envrdist_residuals_t3.pdf`, `FigS9B_lew_length_envrdist_residuals_t3.pdf`, `FigS9C_york_surv_envrdist_residuals_t3.pdf`, `FigS9D_york_length_envrdist_residuals_t3.pdf` |
